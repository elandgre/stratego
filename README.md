# stratego
an AI to play stratego

## Dependencies:
- pytest
- numpy

## Running
The lazy way:
- `make` or `make stratego` will run the backend
- `make test` will run all the tests
- `make clean` will delete all the .pyc files

The python way:
- just the backend
`python game/play.py`

- just the tests
`python -m pytest`
